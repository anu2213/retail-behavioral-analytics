# RetailVision AI

RetailVision AI is an AI-powered retail analytics platform that analyzes customer behavior using computer vision and generates actionable business insights in real time.

## 🌐 Live Demo

Frontend: https://retail-behavioral-analytics.vercel.app/

Explore the deployed application to see the dashboard, analytics, and AI-generated insights in action.

## 🚀 Features

- Real-time face detection using OpenCV
- Age group estimation with InsightFace
- Emotion recognition using DeepFace
- Interactive analytics dashboard
- Foot traffic and customer behavior insights
- AI-generated retail recommendations using Llama 3 (Groq API)
- Session-based analytics tracking
- Export analytics reports as PDF
- Upload video for offline store analysis
- Start a new analysis session anytime

## 🛠 Tech Stack

### Frontend
